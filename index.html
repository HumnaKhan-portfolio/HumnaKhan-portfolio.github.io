Featured project · UTS Tech Festival 2026

TRUST Framework | Legacy Infrastructure Baseline Validation

Developed TRUST, an independent cybersecurity framework exploring baseline validation and operational trust assessment within legacy critical infrastructure environments.

The project investigates how dormant threats, undocumented dependencies, and inherited trust relationships can become normalised within operational baselines, potentially limiting the effectiveness of modern monitoring systems.

Presented as an interactive analyst simulation at UTS Tech Festival 2026, allowing participants to evaluate containment decisions and their operational consequences in a realistic infrastructure scenario.

SSH Brute Force Simulation

Simulated an SSH brute-force attack in a controlled Ubuntu virtual lab and analysed authentication telemetry through Linux log files.

Reconstructed the attack path from username enumeration to successful authentication and mapped the activity to MITRE ATT&CK.
