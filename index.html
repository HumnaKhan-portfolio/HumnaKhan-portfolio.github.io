Featured Project • UTS Tech Festival 2026

TRUST Framework | Legacy Infrastructure Baseline Validation

Developed TRUST, an independent cybersecurity framework exploring baseline validation and operational trust assessment in legacy critical infrastructure environments.

The project looks at how dormant threats, undocumented dependencies, and inherited trust relationships can become normalised inside operational baselines, which can make modern monitoring less effective if the starting point was never properly checked.

Presented as an interactive analyst simulation where participants had to make containment decisions and deal with the operational consequences.

Critical Infrastructure Operational Trust SIEM/SOC Threat Modelling
